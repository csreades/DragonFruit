# Personal bug list

Craig's running list of UX papercuts to fix (not upstream-facing; promote
items to real issues/PRs when they're ready).

## Open

1. **Copy & paste reorganises the layout.** Copy/paste of parts should
   reproduce the copied files and their layout verbatim at the new location —
   it must not re-arrange/re-nest anything.
2. **No drag-select for multiple parts.** Should be able to rubber-band /
   drag-select several parts in the viewport.
3. **Arrangement tools reject negative values.** Array, arrange, and the other
   arrangement tools should accept negative values (spacing, offsets, etc.),
   same as fill-plate already does with negative spacing.
